Tensorflow内存泄漏

Colorjam
2019-05-18
维修指南

用tf经常会出现OOM的现象，查了一下发现了一篇文章How To Debug A Memory Leak In Tensorflow

由于tf存在内存泄漏问题，许多人会用 tcmalloc 来替代 malloc()。

但是运行程序的时候会报错：

踩了一系列坑以后发现，将/usr/lib/libtcmalloc.so.4改为/usr/local/lib/libtcmalloc.so.4即可。
