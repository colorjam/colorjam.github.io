我的XP虚拟机终于可以上网了

Mar 22, 2017

因为我使用的是wifi于是要先在主机的【网络连接】中设置无线网卡属性

在走了很多很多条弯路以后，才发现以前做实验的时候在【编辑】-【虚拟网络编辑器】中设置过VMnet8的IP地址（我的内心是崩溃的

然后发现了很重要的一点，在主机的VMnet8网卡中要设置静态IP地址（这个好像随便设，后面修改成192.168.200.123也是行得通的

然后虚拟机的网络设置自动获取IP地址就可以了

留下了感动的泪水
